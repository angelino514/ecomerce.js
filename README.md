# 🛒 E-Commerce Frontend

Projeto de e-commerce desenvolvido com **HTML, CSS e JavaScript puro**, focado em manipulação do DOM, lógica de frontend e gestão de estado no navegador.

---

## 🚀 Funcionalidades

Este projeto encontra-se em fase inicial e já inclui:

### 🛍️ Carrinho de Compras
- Adição e remoção de produtos
- Atualização de quantidade em tempo real
- Cálculo automático de subtotal, frete e total

### ❤️ Favoritos
- Sistema de lista de produtos favoritos
- Adicionar e remover itens de forma dinâmica

### 🔍 Filtro de Produtos
- Filtragem por categorias
- Atualização dinâmica da interface conforme seleção

### ⚡ Atualização em Tempo Real
- Interface atualizada automaticamente ao alterar dados
- Sem necessidade de recarregar a página

### 🧠 Manipulação do DOM
- Criação dinâmica de elementos com JavaScript
- Componentização da interface em funções reutilizáveis

### 💾 Persistência de Dados
- Utilização de `localStorage`
- Carrinho e favoritos mantidos após recarregar a página

---

## 🧱 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage API
- Manipulação avançada do DOM

## 🎯 Objetivo do Projeto

Este projeto tem como objetivo desenvolver habilidades em:

- Lógica de programação aplicada ao frontend

- Simulação de um sistema de e-commerce funcional
